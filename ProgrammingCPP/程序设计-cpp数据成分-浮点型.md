# 数据成分-浮点型（实型）

## 1、定义
计算机系统中浮点型也称实型，通常指数学概念中的小数。

## 2、精度
精度位数：整数位数+小数位数=有效精度位数。系统默认打印是6位有效精度位数，可以通过操作运算符打印更多的精度位数。

## 3、单精度
定义规范：
```c++
type variable_name = value;
```
demo:
```c++
float f1;
f1 = 3.14;
float f2 = 3.14;
```
